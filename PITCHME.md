# Webサイトの概要
## Webページを構成する要素
### ソフトウェア
1. ブラウザ
	1. HTML
	2. CSS
	3. JavaScript
	4. 画像
	5. 動画


### ハードウェア
1. サーバ（Webサーバ）
2. DNSサーバ

### 通信方法（プロトコル）
1. HTTP

## Webページが表示されるまで
ブラウザ  
　↓  （URLの問い合わせをする）  
DNSサーバ  
　↓  （URLのIPアドレスが送られてくる）  
ブラウザ  
　↓  （DNSサーバから送られてきたIPを持つサーバへデータのリクエストをする）  
サーバ  
　↓ （HTMLやその中で使用するアセットが送られてくる）  
ブラウザ
